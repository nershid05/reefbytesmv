# ReefbytesMV Website

## Overview
A static HTML/CSS website for ReefbytesMV, a software engineering and solutions company.

## Project Structure
- `index.html` — Main homepage
- `careers.html` — Careers page
- `styles.css` — Global stylesheet

## Running the Project
Served with Python's built-in HTTP server on port 5000:
```
python3 -m http.server 5000
```

## Deployment
Configured as a static site deployment with the root directory (`.`) as the public directory.
